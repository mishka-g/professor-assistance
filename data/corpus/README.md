# Corpus — the professor's published work

Drop the professor's **published** papers here (`.pdf`, `.docx`, or `.txt`).
These are used only to *ground* the reviewer (terminology, phrasing, conventions).

- 20–30 papers is a good start; more is better.
- This repo ships with open-access arXiv preprints for out-of-the-box testing.
- After adding or removing files, run: `profa ingest`
